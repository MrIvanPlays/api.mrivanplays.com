MrIvanPlays API

List of API requests available

It looks like you've hit a request which doesn't exist or typed only "api.mrivanplays.com". That's why we're listing all available requests down bellow

Enable white mode


https://api.mrivanplays.com/spigot/download/[resource id] - downloads the spigot resource with the specified id (latest version). This is a thread-blocking request! Run it always on asynchronous threads.

Errors for /spigot/download/ : {"error": 404, "message": "Invalid resource id '[insert invalid resource id here]'" ; {"error": 404, "message": "Resource cannot be found on spigot"}


https://api.mrivanplays.com/meme - gets a random meme from reddit. Returned content is a json string with a link to the image.


https://api.mrivanplays.com/library/version/[library name] - Shows the version of the specified library, which have been uploaded to the specified repo url with the specified group id. The returned content is a json string with a version string and a success boolean.

Errors for /library/version/ : {"success": false, "error": 404, "message": "Dependency not found"}

Query params for /library/version/ : repoUrl - a url to repository , can also be "mavenCentral" ; groupId - example "com.mrivanplays" ; raw - should the output not be a json string, but a version ; If no query parameters specified, then the default repo url is https://repo.mrivanplays.com/ and the default group id com.mrivanplays

Example url: https://api.mrivanplays.com/library/version/gson?repoUrl=mavenCentral&groupId=com.google.code.gson
