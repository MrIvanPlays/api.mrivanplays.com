MrIvanPlays API

List of API requests available

It looks like you've hit a request which doesn't exist or typed only "api.mrivanplays.com". That's why we're listing all available requests down bellow

TEMPORARILY DISABLED

https://api.mrivanplays.com/spigot/download/[resource id] - downloads the spigot resource with the specified id (latest version). This is a thread-blocking request! Run it always on asynchronous threads.
A successful request will give you a input stream of the jar.
Error responses: 
{"error": 404, "message": "Invalid resource id '[insert invalid resource id here]'"}
{"error": 404, "message": "Resource cannot be found on spigot"}
{"error": 403, "message": "Resource is premium"}
{"error": 403, "message": "Cannot download resource which has external download link."}
{"error": 403, "message": "Resource name invalid."}

https://api.mrivanplays.com/meme - gets a random meme from reddit.

Query parameters:
raw - if true, the output won't be a json string, but a direct version

Examples of successful requests:

Example request url: https://api.mrivanplays.com/meme
The specified link will output this: 

{"image": "[link to the image, video, etc]"}
Example request url: https://api.mrivanplays.com/meme?raw=true
The specified link will output this: 
[link to the image, video, etc]

https://api.mrivanplays.com/library/version/[library name] - Shows the version of the specified library, which have been uploaded to the specified repo url with the specified group id.

Query parameters:
repoUrl - a url to repository , can also be "mavenCentral" for maven central repository ;
groupId - example "com.mrivanplays" ;
raw - if true, the output won't be a json string, but a direct version ;

If no query parameters specified, then the default repo url is https://repo.mrivanplays.com/repository/ivan/ and the default group id com.mrivanplays

Examples of successful requests:

Example request url: https://api.mrivanplays.com/library/version/gson?repoUrl=mavenCentral&groupId=com.google.code.gson
The specified link will output this: 

{"success": true, "version": "[version]"}
Example request url: https://api.mrivanplays.com/library/version/gson?repoUrl=mavenCentral&groupId=com.google.code.gson&raw=true
The specified link will output this: 
[version]

Error responses: 
{"success": false, "error": 404, "message": "Dependency not found"}

DISCLAIMER: This API will be shut down at 1st of January, 2020

https://api.mrivanplays.com/trees - retrieves the count of trees that are being planted by https://teamtrees.org/

Query parameters:
raw - if true, the output won't be a json string, but a direct version

Examples of successful requests:

Example request url: https://api.mrivanplays.com/trees
The specified link will output this: 
{"success": true, "trees": "[number of planted trees]"}
Example request url: https://api.mrivanplays.com/trees?raw=true
The specified link will output this: 
[number of planted trees]

Error responses: 
{"success": false, "error": 503, "message": "teamtrees.org is being overloaded with traffic"}
{"success": false, "error": 408, "message": "teamtrees.org timed out"}
