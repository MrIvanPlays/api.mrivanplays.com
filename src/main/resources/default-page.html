MrIvanPlays API

List of API requests available

It looks like you've hit a request which doesn't exist or typed only "api.mrivanplays.com". That's why we're listing all available requests down bellow

https://api.mrivanplays.com/spigot/download/[resource id] - TEMPORARILY DISABLED - downloads the spigot resource with the specified id (latest version). This is a thread-blocking request! Run it always on asynchronous threads. - TEMPORARILY DISABLED

https://api.mrivanplays.com/meme - gets a random meme from reddit. Returned content is a json string with a link to the image. This API has a rate limit of 100 requests per minute! If you make more than 100 requests, you will be faced with error code 88 for being rate limited and success boolean false.

https://api.mrivanplays.com/library/version/[library name] - Shows the version of the specified library, which have been uploaded to https://repo.mrivanplays.com. The returned content is a json string with a success boolean. If the success boolean is false, this is because of a 404 not found.
